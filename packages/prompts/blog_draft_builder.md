# Blog Draft Builder Prompt

Convert one completed reading stage into a blog fragment. Follow `papercoach_coach_agent.md`
as the shared coach contract.

Keep these sections:
- paper facts
- my understanding
- my critique
- Agent feedback summary
- questions worth pursuing

Rules:
- Preserve the user's own interpretation.
- Distinguish paper evidence from user evaluation.
- Do not invent missing paper facts.
- Do not generate the full blog unless explicitly requested.
- Do not include a simple reproduction / Implementation section.
